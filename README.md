# 🚀 DSA-Projects

A collection of Data Structures and Algorithms (DSA) projects implemented in Java. This repository demonstrates the practical application of graphs, pathfinding, game development, and problem-solving techniques.

## 📂 Projects Included

### 🐍 Snake Game
- Classic Snake Game implementation in Java.
- Features smooth movement, score tracking, and collision detection.

**Folder:** `Snake-Game`

---

### 🚇 Metro App
- A Java-based Metro Route Finder.
- Uses Graphs and Dijkstra's Algorithm to find the shortest path between stations.

**Folder:** `THE-METRO-APP`

---

### 💣 Bomberman
- A Java implementation of the classic Bomberman game.
- Includes player movement, bombs, enemies, explosions, and game logic.

**Folder:** `bomberman`

---

### 🏫 Campus Navigation System
- Navigation system for finding the shortest route inside a campus.
- Uses Graphs and Dijkstra's Algorithm.

**Folder:** `campus-navigation-system`

---

### 🚉 Metro DSA
- Metro route management project using Graph Data Structure.
- Calculates shortest distance and optimal routes between metro stations.

**Folder:** `metro DSA`

---

### 🧩 Sudoku Solver
- Java implementation of a Sudoku Solver.
- Uses Backtracking Algorithm to solve Sudoku puzzles efficiently.

**Folder:** `sudoku_project`

---
